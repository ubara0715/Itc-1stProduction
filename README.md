## ゲームストーリー
妖怪たちと共存する巫女である主人公は、突如暴走し始めた妖怪たちを鎮めるため、<br>
御札で対峙しながら妖怪たちが守る神社へと突入する…！

## ゲームルール
妖怪たちを全員倒し、最奥へ進め！
苦手な属性で攻撃しなければ、妖怪たちに妖術をかけられデバフをくらう！<br>
うまく切り替えて戦い、制限時間以内に暴走の原因を突き止めよう！
